# Development-of-a-Sensor-Fusion-Model-for-Advanced-Driver-Assistance-System-Applications

Honours Thesis – University of Sydney

This project develops an Advanced Driver Assistance System (ADAS) prototype using radar and ultrasonic sensors. The system performs real-time obstacle detection and collision avoidance through sensor fusion techniques.

Technologies:
- Python
- Embedded systems
- Radar & ultrasonic sensors
- Sensor fusion algorithms

Author: Imran Tauqeer
